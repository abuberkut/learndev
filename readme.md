1) Установите OpenServer
2) Разместите папку learndev в папку C:\OSPanel\domains
3) Создайте базу данных 'learndev_db' и импортируйте 
	learndev_db.sql туда.
4) Запустите OpenServer.exe из папки C:\OSPanel
5) В нижнем правом углу появится флаг, запустите его.
	Оно статен зеленым
6) Нажмите на флаг, в пункте "мои сайты" выберите learndev
